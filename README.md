# ⚔️ Demon Slayer RPG

A console-based RPG inspired by **Demon Slayer: Kimetsu no Yaiba** built using C++ and OOP concepts.

This project started as a semester project and was gradually expanded with additional mechanics and story elements to make the gameplay more engaging.

Players can choose between **Tanjiro**, **Zenitsu**, and **Inosuke**, each with their own abilities and playstyle, and progress through multiple acts while fighting demons and eventually facing **Muzan** in the final battle.

---

## Features

- 3 playable characters with unique abilities
- Turn-based combat system
- Breathing techniques for each character
- Inventory and equipment system
- Crow Token shop system
- Wisteria potions for healing
- Critical hits and dodge mechanics
- Multiple demon encounters and boss fights
- Multiple endings based on player performance and final score

---

## Characters

### 🌊 Tanjiro
Balanced character with Water Breathing abilities.

### ⚡ Zenitsu
Lower health but high burst damage with Thunderclap Flash.

### 🐗 Inosuke
Higher health and an aggressive fighting style with double attacks.
